# Personal Finance Tracker (CLI App)

A beginner-friendly Python CLI app to log expenses and save them into a `.csv` file. Built while learning Python through the 100 Days of Code Bootcamp.

## Features
- Add an expense: name, amount, category, date
- Saves to `expenses.csv` for easy tracking
- Automatically uses today’s date if left blank

## Tech Used
- Python
- Built-in `csv` and `datetime` modules
